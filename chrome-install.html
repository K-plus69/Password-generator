How to install PG-E on Google Chrome

Download pg-e.zip here

1. Download the PG-E .zip file and right click it after the download finished.

2. Select "extract all"

3. A Pop-up will come up and select a Safe location where the folder wont be deleted.

4. Click "extract"

5. Open Chrome and go to chrome://extensions/ or... (picture)

6. Toggle the "Developer mode" on and click "Load unpacked"

7. Select the PG-E folder

8. Pin PG-E to be in the status bar

Finished! PG-E is good to go!
